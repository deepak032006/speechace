## Speechace Speaking Test Report (Clone)

### Tech Used
- HTML, CSS, JavaScript
- Node.js + Express (Backend)

### How to Run
1. cd backend
2. npm install
3. node server.js
4. Open frontend/index.html in browser

### Data Source
Scores are stored in backend/data.json

### Feedback Logic
Feedback automatically updates based on score values fetched from backend API.
